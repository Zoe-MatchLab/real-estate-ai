```
│ 新建任务                           [保存]│

│           选择任务类型                    │
│ ┌─────────┐ ┌─────────┐               │
│ │  🔴探盘 │ │ 🟣话术 │               │
│ │  线下楼盘│ │ AI模拟 │               │

│ │ 🟢内容 │ │ 🔴新盘 │               │
│ │  推广文案│ │ 上线准备│               │
│ └─────────┘ └─────────┘               │
│                                         │
│           快捷创建                       │
│ ┌─────────────────────────────────────┐│
│ │ 📋 谈判任务                         ││
│ ├─────────────────────────────────────┤│
│ │ 📍 立即探盘                         ││
│ ├─────────────────────────────────────┤│
│ │ 🎤 开始训练                         ││
│ └─────────────────────────────────────┘│
└─────────────────────────────────────────┘
```

| 评审项 | 状态 | 建议 | 负责人 |
|--------|------|------|---------|
| 类型选择布局 | ✅ | 2x2 网格 | - |
| 快捷入口必要性 | ⬜ | 确认快捷创建是否需要 | 产品 |
| 新建流程合理性 | ⬜ | 确认是否需要先选类型 | 产品 |

**产品评审意见：**
1. **快捷创建是否需要？**
   - 谈判任务 → 跳转到谈判表单
   - 立即探盘 → 跳转到探盘执行
   - 开始训练 → 跳转到话术训练
   - 建议：保留，因为用户经常需要快速访问

2. **新建流程是否合理？**
   - 当前：选择类型 → 选择楼盘 → 填写配置
   - 建议：确认是否符合实际使用场景

**UX 评审意见：**
> 页面布局和交互是否流畅？
