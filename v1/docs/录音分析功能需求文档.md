> 版本：V1.0

> 状态：待UI/UX Review

---

## 一、功能概述

### 1.1 功能定位

约看录音分析是经纪人上传电话邀约录音，AI分析录音内容，给出话术改进建议的能力提升工具。

### 1.2 业务价值

- 帮助经纪人了解自己的话术水平
- 识别客户需求和意向
- 获得专业改进建议提升成交率

## 二、PC端 - 录音分析管理后台

### 2.1 页面路径

`/v1/pc/record/record-list.html`（已存在部分原型）

### 2.2 功能模块

#### 2.2.1 录音列表页

| 功能 | 说明 |
| --- | --- |
| 列表展示 | 展示所有经纪人的录音记录 |
| 筛选 | 按楼盘/经纪人/分析状态/时间筛选 |
| 搜索 | 关键词搜索录音名称/经纪人 |
| 查看详情 | 点击查看分析报告 |
| 导出 | 导出录音列表/分析报告 |

**列表字段**：
| 字段 | 说明 |
| --- | --- |
| 录音名称 | 文件名 |
| 关联楼盘 | 关联的楼盘名称 |
| 经纪人 | 上传人姓名 |
| 上传时间 | 上传时间 |
| 分析状态 | 待分析/分析中/已分析/分析失败 |
| 综合评分 | 1-100分 |

#### 2.2.2 分析报告详情页

| 功能 | 说明 |
| --- | --- |
| 录音播放 | 在线播放录音 |
| 报告查看 | 查看AI分析结果 |
| 报告导出 | 导出分析报告 |

**分析结果内容**：
| 维度 | 说明 |
| --- | --- |
| 综合评分 | 1-100分 |
| 话术技巧 | 开场白、需求挖掘、异议处理、促成技巧 |
| 情绪判断 | 客户意向度（高/中/低） |
| 客户需求 | 客户关注点、潜在需求、购买意向 |
| 改进建议 | 具体可操作的提升建议 |

#### 2.2.3 提示词配置

| 功能 | 说明 |
| --- | --- |
| 提示词模板 | 配置AI分析提示词 |
| 预览 | 预览分析报告样式 |

#### 2.2.4 数据统计

| 功能 | 说明 |
| --- | --- |
| 上传量统计 | 按日/周/月统计录音上传量 |
| 分析量统计 | 按日/周/月统计分析量 |

## 三、小程序端 - 录音分析

### 3.3 功能模块

#### 3.3.1 录音列表页

**入口**：
- 工具列表页 → 约看录音分析

**列表字段**：
| 字段 | 说明 |
| --- | --- |
| 录音名称 | 文件名或自动命名 |

## 五、状态说明

### 5.1 录音状态

| 状态 | 说明 |
| --- | --- |
| 待分析 | 已上传，等待分析 |
| 分析中 | AI正在分析中 |
| 已分析 | 分析完成，可查看结果 |
| 分析失败 | 分析失败，可重新上传 |

### 5.2 状态流转

```
上传录音 → 待分析 → 分析中 → 已分析
                      ↓
                  分析失败 → 重新上传
```
