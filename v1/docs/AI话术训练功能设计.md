## 1. 产品概述

### 1.1 功能定位

AI话术训练是一款基于AI实时对话的培训工具，帮助经纪人在仿真场景中练习话术，提升邀约、接待、讲解等业务能力。

**与新房作业的关系**：
- 新房上线时，系统自动关联该楼盘对应的训练项目
- 探盘任务下发后，经纪人可在任务卡片点击「产品力训练」直接进入训练
- 话术训练是**通用功能**，不涉及任务下发和审核流程

### 1.2 核心价值

- **标准化SOP**：通过AI模拟真实客户对话，帮助经纪人掌握标准话术
- **随时练习**：打破时间空间限制，利用碎片时间训练
- **即时反馈**：AI实时分析对话，给出改进建议

### 1.3 与新房项目的关联

```
新房上线
    ↓
自动关联该楼盘的训练项目（预配置）
    ↓
探盘任务下发
    ↓
经纪人收到通知 + 训练入口提醒
    ↓
点击任务卡片「产品力训练」→ 进入AI话术训练
```

## 2. 用户角色

| 角色        | 操作             |
| --------- | -------------- |
| 总部干部      | 配置训练项目、场景、话术模板（PC管理后台） |
| 业务总监/区域经理 | 查看训练数据（PC管理后台）  |

**说明**：
- 话术训练任务**不需要下发**，是经纪人自主练习的通用功能
- 新房上线时自动关联训练项目，属于楼盘知识体系的一部分

## 3. 任务状态定义

### 3.1 状态说明

| 状态 | 说明 | 卡片展示 |
|------|------|----------|
| 待训练 | 训练项目已关联，尚未开始 | 「开始训练」按钮 |
| 已完成 | 已完成至少一次训练 | 显示完成次数和评分 |

### 3.2 状态流转

```
待训练 ──→ 已完成（完成训练后）
    ↑
    └──────（可再次训练，每次训练都记录）
```

**说明**：
- 同一训练项目可重复练习，每次练习都会记录
- 不存在"驳回"或"审核"概念
- 训练是经纪人的自主行为，不涉及考核

```
│                      AI话术训练流程                         │
└─────────────────────────────────────────────────────────────┘

  ┌──────────────┐
  │  1. 选择项目  │ ← 从官方推荐/我创建的中选择

  └──────┬───────┘
         ↓

  │  3. 选择模式  │ ← 语音通话 / 打字聊天 / 语音+文字
  └──────┬───────┘
         ↓
  ┌──────┴───────┐
  │  4. 开始训练  │ ← AI扮演客户，经纪人练习对话

         ↓
  ┌──────┴───────┐
  │  5. 训练完成  │ ← AI给出评价和建议，状态变为「已完成」
  └──────────────┘
```

## 5. 交互模式说明

### 5.1 语音通话模式

- **交互**：经纪人和AI进行实时语音对话
- **场景**：模拟电话邀约，AI扮演客户接听
- **UI表现**：
  - 显示通话时长
  - 语音波形动画
  - 录音按钮

### 5.2 打字聊天模式

- **交互**：经纪人打字输入，AI实时回复
- **场景**：需要思考的话术练习
- **UI表现**：
  - 对话气泡形式（类似微信）
  - AI消息在左，经纪人消息在右

### 5.3 语音+文字模式

- **交互**：语音对话 + 文字同步显示
- **场景**：最完整的训练体验
- **UI表现**：
  - 上方显示对话文字
  - 下方显示语音波形
  - 支持语音输入

## 6. 项目配置说明（管理后台）

> 以下为后台功能，前端展示数据来源

### 6.1 训练项目

| 字段   | 说明                  |
| ---- | ------------------- |
| 项目名称 | 如：中海汇德里新盘训练         |
| 关联楼盘 | 关联的新房项目（新房上线时自动关联） |
| 场景类型 | 邀约/接待/沙盘讲解/样板间讲解/逼定 |
| 话术模板 | AI扮演客户的角色设定和话术      |
| 评分规则 | 关键词命中/语义分析规则        |

**与新房项目的关联说明**：
- 新房项目审核通过上线时，系统自动关联预配置的训练项目
- 训练项目不需要单独下发，是楼盘知识体系的一部分
- 经纪人在探盘任务中可点击「产品力训练」直接进入练习

### 6.2 角色配置

| 角色  | 话术设定      |
| --- | --------- |
| 客户A | 首次看房，关注价格 |
| 客户B | 投资客，关注回报率 |
| 客户C | 自住客，关注配套  |
