> 文档版本：V1.0

> 状态：待评审

---

## 一、产品定位

### 1.1 模块定位

新盘管理是房产AI作业平台的核心业务模块，统一管理所有新房项目的全生命周期信息，包含楼盘资料管理、审核流程、物料管理三大块，并支持与探盘任务联动。

### 1.2 双端定位

| 端 | 定位 | 核心用户 |
|----|------|----------|
| PC管理后台 | 楼盘资料录入、审核、物料管理 | 运营人员、审核人员、管理员 |
| 小程序 | 楼盘信息查看、推广、联系 | 经纪人 |

### 1.3 核心价值

| 价值 | 说明 |
|------|------|
| 统一管理 | 避免楼盘数据分散，信息集中管理 |
| 审核合规 | 标准化录入+审核流程，保证资料完整性 |
| 物料关联 | 楼盘资料自动关联，方便推广使用 |
| 探盘联动 | 审核通过后自动生成探盘任务 |
| 移动推广 | 经纪人随时查看楼盘，一键推广 |

---

## 二、用户角色

| 角色 | PC端操作 | 小程序端操作 |
|------|----------|-------------|
| 经纪人 | - | 查看楼盘、收藏、分享、联系 |
| 运营人员 | 录入楼盘、提交审核、物料管理 | - |
| 审核人员 | 审核楼盘通过/驳回 | - |
| 管理员 | 全局管理、配置 | - |

## 七、联动关系

### 7.1 新盘与探盘任务联动

```
新盘审核通过
    ↓
自动生成探盘任务（待分发状态）
    ↓
运营人员配置执行人员后下发
    ↓
经纪人收到任务通知
    ↓
执行探盘任务
    ↓
提交审核
    ↓
审核通过 → 探盘资料自动同步到楼盘物料
```

### 7.2 物料共享

```
探盘任务完成
    ↓
采集的物料自动同步到楼盘详情
    ↓
经纪人可在小程序查看使用
```
