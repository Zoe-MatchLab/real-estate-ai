## 一、产品概述

### 1.1 背景与目标

房产经纪人日常需要接收并执行多种任务：探盘打卡、话术训练、内容发布等。当前各任务分散在不同入口，缺乏统一的通知和执行入口。

**本期目标：** 在小程序侧建立统一的任务消息通知和执行流程，覆盖探盘打卡、话术训练、内容发布三种任务类型，支持项目制任务管理。

### 1.2 核心场景

| 场景 | 描述 |
|------|------|
| 接收任务 | PC管理后台下发的任务自动推送到小程序 |
| 消息通知 | 首页顶部通知入口，聚合所有任务消息 |
| 执行任务 | 经纪人执行探盘打卡、话术训练等任务 |
| 进度追踪 | 下发人可跟进各阶段完成情况（后续规划） |

### 1.3 任务类型

| 类型 | 说明 | 执行方式 |
|------|------|----------|
| 🏠 探盘打卡 | 到楼盘现场拍照、打卡、采集信息 | 4阶段分步引导 |
| 💬 话术训练 | AI实时对话练习话术 | 跳转speech-train.html |
| 📝 内容发布 | 基于探盘素材生成文案/视频 | 提示暂未开放 |

## 四、功能详述

### 4.1 首页通知入口

**位置：** tools.html 顶部工具栏

**样式：**
```
┌─────────────────────────────────────┐
│ 🔔 通知(3)              🔍        │ ← 通知铃铛 + 数字红点
├─────────────────────────────────────┤
│                                     │
│         AI工具网格                  │
│                                     │
└─────────────────────────────────────┘
```

**交互说明：**
- 铃铛图标 + 红色数字：表示未读消息数量
- 点击铃铛 → 展开通知面板弹窗
- 新任务到达 → 自动弹出通知提示（2秒后消失）

---

### 4.2 通知面板

**形式：** 底部全宽弹窗，不跳转页面

**布局：**
```
┌─────────────────────────────────────┐
│ 📋 通知               [清空全部]  × │ ← 标题栏
├─────────────────────────────────────┤
│ [全部] [探盘] [话术] [内容]         │ ← 分类Tab
├─────────────────────────────────────┤
│ 🔴 中海汇德里探盘任务               │
│    2/4阶段进行中，请注意完成          │ ← 未读（红色圆点）
│    刚刚                        →   │
├─────────────────────────────────────┤
│ ⚪ 万科城市之光话术训练             │
│    新任务已下发，请及时完成            │ ← 已读（灰色圆点）
│    10分钟前                    →   │
├─────────────────────────────────────┤
│ ⚪ 朋友圈文案生成                   │
│    等待探盘完成后执行                  │
│    昨天                        →   │
└─────────────────────────────────────┘
```

**功能说明：**

| 功能 | 说明 |
|------|------|
| 分类筛选 | 全部/探盘/话术/内容 四个Tab |
| 时间排序 | 最新消息在前 |
| 已读/未读 | 圆点颜色区分（红=未读，灰=已读） |
| 标记已读 | 点击消息后变为已读状态 |
| 清空全部 | 一键清除所有通知 |
| 关闭面板 | 点击×或遮罩层关闭 |

**消息卡片信息：**

| 字段 | 说明 |
|------|------|
| 任务图标 | 🔴/⚪ 圆点表示未读/已读 |
| 任务名称 | 任务标题 |
| 消息摘要 | 任务状态或提醒内容 |
| 发布时间 | 刚刚/10分钟前/昨天 等相对时间 |
| 跳转入口 | → 箭头，点击进入详情 |

### 4.4 探盘打卡执行流程

#### 4.4.1 四阶段说明

```
┌─────────────────────────────────────────────────────────────┐
│                    探盘任务四阶段流程                         │
└─────────────────────────────────────────────────────────────┘

  阶段1          阶段2          阶段3          阶段4
┌────────┐    ┌────────┐    ┌────────┐    ┌────────┐
│定位打卡│ → │采集执行│ → │文案确认│ → │成果生成│
└────────┘    └────────┘    └────────┘    └────────┘
    │              │              │              │
    ↓              ↓              ↓              ↓
 到达现场      拍照/视频      确认文案        封面+视频
 签到确认      /录音/文本     脚本内容        提交审核
                            (后台生成)     (后台生成)
```

**阶段详情：**

| 阶段 | 名称 | 必填 | 说明 |
|------|------|------|------|
| 阶段1 | 定位打卡 | ✅ | 到达楼盘现场，定位签到 |
| 阶段2 | 采集执行 | ✅ | 后台配置点位（拍照/视频/录音/文本），不完成不能提交 |
| 阶段3 | 文案确认 | - | 后台生成文案/脚本，经纪人确认 |
| 阶段4 | 成果生成 | - | 后台基于确认内容生成封面图+短视频 |

#### 4.4.2 页面布局（task-detail.html）

```
┌─────────────────────────────────────────┐
│ ← 探盘任务详情                    ···   │
├─────────────────────────────────────────┤
│                                         │
│  🏠 中海汇德里                           │
│  📍 浦东新区张江高科技园区                │
│                                         │
│  ━━━━━━━━━━━━━━ 2/4 ━━━━━━━━━━━━━━   │
│                                         │
│  【阶段1】✅ 定位打卡                    │
│  ┗━ 到达现场，已签到  10:30:00          │
│                                         │
│  【阶段2】🔄 采集执行                    │
│  ┗━ 必填：沙盘照片    (0/1)  📷添加    │
│  ┗━ 必填：样板间视频  (0/1)  🎬添加    │
│  ┗━ 选填：周边配套    (0/3)  ➕添加    │
│                                         │
│  【阶段3】⬜ 文案确认                    │
│  ┗━ 后台生成中...                      │
│                                         │
│  【阶段4】⬜ 成果生成                    │
│  ┗━ --                                 │
│                                         │
├─────────────────────────────────────────┤
│                                         │
│  [           提交审核           ]        │ ← 底部按钮
│  (灰色/不可点，需完成必填项)              │
│                                         │
└─────────────────────────────────────────┘
```

**采集点位类型：**

| 类型 | 图标 | 说明 |
|------|------|------|
| 拍照 | 📷 | 拍照上传图片 |
| 视频 | 🎬 | 录制短视频 |
| 录音 | 🎤 | 录制语音 |
| 文本 | 📝 | 填写文字信息 |

**阶段状态显示：**

| 状态 | 图标 | 颜色 |
|------|------|------|
| 已完成 | ✅ | 绿色 |
| 进行中 | 🔄 | 橙色 |
| 待执行 | ⬜ | 灰色 |

### 4.5 话术训练跳转

**规则：** 话术训练任务可独立执行，不依赖探盘打卡完成。

**交互流程：**
```
通知消息/任务卡片 → 点击话术训练 → speech-train.html
                                         │
                                    选择项目
                                         │
                                    选择模式
                                         │
                                   开始训练
```

---

### 4.6 内容发布

#### 4.6.1 解锁条件

内容发布依赖探盘打卡（阶段2）完成，后台根据采集的素材生成文案和视频。

```
探盘打卡(阶段2)完成 → 内容发布解锁 → 可执行
         │
         ↓ 未完成
    内容发布锁定 → 提示"需先完成探盘打卡"
```

#### 4.6.2 各状态提示

| 状态 | 提示文案 | 交互 |
|------|----------|------|
| 待解锁 | "内容发布需先完成探盘打卡" | 点击弹出提示 |
| 待执行 | "请确认文案脚本后生成内容" | 可进入 |
| 进行中 | "正在生成封面和短视频..." | 等待 |
| 已完成 | "内容已提交审核" | 查看结果 |

## 六、交互汇总

| 入口 | 行为 |
|------|------|
| 首页通知铃铛 | 展开通知面板弹窗 |
| 通知面板分类Tab | 筛选对应类型消息 |
| 通知消息 | 标记已读 + 跳转任务卡片 |
| 清空全部 | 清除所有通知 |
| 探盘任务卡片 | 进入task-detail.html |
| 探盘打卡入口 | 进入4阶段执行流程 |
| 话术训练入口 | 跳转speech-train.html |
| 内容发布（已解锁） | 跳转内容发布页（本期提示暂未开放） |
| 内容发布（未解锁） | 弹出提示"需先完成探盘打卡" |
