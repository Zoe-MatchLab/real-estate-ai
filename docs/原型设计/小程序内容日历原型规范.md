## 二、页面结构

### 2.1 布局层次

| 层次 | 组件 | 说明 |
|------|------|------|
| 1 | Header | 标题栏 + 通知入口 |
| 2 | 日历组件 | 可折叠的月份日历 |
| 3 | 分类Tab | 任务类型筛选 |
| 4 | 任务列表 | 按日期分组展示 |
| 5 | TabBar | 底部导航（4个首页）|

### 2.2 移除项

- ❌ task-list 相关元素（已有任务卡片）
- ❌ 独立的列表切换按钮

## 三、通知面板交互

### 3.1 触发方式

| 触发 | 动作 |
|------|------|
| 点击铃铛 | 底部面板从屏幕底部向上滑出 |
| 点击遮罩 | 面板收回 |
| 点击关闭按钮 | 面板收回 |
| 滑动面板向下 | 面板收回 |

### 3.2 面板内容

```
┌─────────────────────────────────────────────────────────────┐
│  通知                    [清空]              [✕]            │
├─────────────────────────────────────────────────────────────┤
│  [全部] [探盘] [话术] [内容]                                 │
├─────────────────────────────────────────────────────────────┤
│  ● 中海汇德里新盘探盘                                       │
│    您有一个新任务待执行                      10分钟前        │
│                                                             │
│  ○ 万科翡翠公园新盘上线                                     │
│    任务即将开始，请准备物料                  1小时前         │
│                                                             │
│  ● 议价谈判话术练习                                         │
│    新训练项目已上线                          昨天           │
└─────────────────────────────────────────────────────────────┘

说明：
● = 未读通知（橙色圆点 + 浅橙色背景）
○ = 已读通知（灰色圆点 + 白色背景）
```

### 3.3 通知项字段

| 字段 | 展示 | 说明 |
|------|------|------|
| 未读标记 | 橙色圆点 | 左侧8px圆点 |
| 通知名称 | 14px深色粗体 | 任务名称 |
| 通知内容 | 12px浅色文字 | 简短描述 |
| 发布时间 | 12px灰色 | 相对时间 |
| 点击效果 | scale(0.98) + 背景变深 | 点击反馈 |

### 3.4 分类Tab

| Tab | 筛选内容 |
|-----|----------|
| 全部 | 所有通知 |
| 探盘 | type=house |
| 话术 | type=speech |
| 内容 | type=content |

## 四、任务卡片设计

### 4.1 任务类型定义

| 类型 | 图标 | 颜色 | 说明 |
|------|------|------|------|
| **探盘** | 📍 | 橙色 #FA8C16 | 实地勘盘任务 |
| **话术** | 💬 | 紫色 #722ED1 | 话术训练任务 |
| **内容** | 📝 | 蓝色 #007AFF | 内容发布任务 |
| **新盘** | 🏠 | 绿色 #00B42A | 新盘上线任务 |

### 4.2 卡片基础信息

```
┌─────────────────────────────────────────────────────────────┐
│  [类型图标] [类型标签]          [执行人头像] [时间状态]      │
│                                                             │
│  任务标题（14px粗体）                                       │
│  任务副标题/说明（12px灰色）                                │
│                                                             │
│  [进度条区域 - 仅探盘任务有]                                 │
│  ████████████░░░░░░░░░░░░░░░░░░░░░░░░░░  65%             │
│                                                             │
│  [生成中状态 - 仅生成阶段显示]                               │
│  ⏳ 阶段3 文案生成中 45%...                                 │
│     可返回任务列表，后台持续运行                             │
└─────────────────────────────────────────────────────────────┘
```

### 4.3 卡片字段说明

| 字段 | 位置 | 样式 | 示例 |
|------|------|------|------|
| 类型图标 | 左上 | 24px圆形+图标 | 📍 💬 📝 🏠 |
| 类型标签 | 图标右侧 | 12px标签 | 探盘 / 话术 / 内容 / 新盘 |
| 执行人 | 右上组 | 24px圆形头像叠放 | 张三(张) 李四(李) |
| 时间状态 | 执行人右 | 12px + 颜色 | 今日18:00 / 明天10:00 / 已完成 |
| 任务标题 | 中部 | 14px粗体深色 | 中海汇德里新盘探盘 |
| 任务副标题 | 标题下 | 12px灰色 | 共4个打卡点·已完成2个 |
| 进度条 | 副标题下 | 高度8px圆角条 | ████░░░░ 65% |
| 生成状态 | 进度条下 | 12px+spinner | ⏳ 阶段3 文案生成中... |

### 4.4 时间状态颜色

| 状态 | 颜色 | 说明 |
|------|------|------|
| 紧急 | 红色 #FA5151 | 即将到期（<2小时）|
| 警告 | 橙色 #FF7D00 | 今天内到期 |
| 正常 | 灰色 #86909C | 未来时间 |
| 已完成 | 绿色 #00B42A | 任务完成 |

### 4.5 进度条颜色

| 类型 | 颜色 |
|------|------|
| 探盘 | 橙色 #FA8C16 |
| 话术 | 紫色 #722ED1 |
| 内容 | 蓝色 #007AFF |
| 新盘 | 绿色 #00B42A |

## 五、日历组件设计

### 5.1 布局

```
┌─────────────────────────────────────────────────────────────┐
│  <  2026年4月  >                                            │
│  ┌────┬────┬────┬────┬────┬────┬────┐                        │
│  │日  │一  │二  │三  │四  │五  │六  │                        │
│  ├────┼────┼────┼────┼────┼────┼────┤                        │
│  │    │    │    │ ① │ ② │ ③ │ ④ │  ← 数字为有任务的日期   │
│  │    │    │    │ ● │ ● │    │    │  ← 红点标记有任务      │
│  └────┴────┴────┴────┴────┴────┴────┘                        │
└─────────────────────────────────────────────────────────────┘
```

### 5.2 交互

| 交互 | 动作 |
|------|------|
| 点击 < | 上月 |
| 点击 > | 下月 |
| 点击日期 | 选中 + 滚动到该日期任务 |
| 有任务日期 | 右下角红点 |
