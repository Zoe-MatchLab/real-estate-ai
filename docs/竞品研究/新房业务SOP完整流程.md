## 一、新房作业PDCA循环概述

传统新房作业是一套**阶梯式上升**的循环体系：

```
┌─────────────────────────────────────────────────────────┐
│                  新房作业PDCA循环                       │
└─────────────────────────────────────────────────────────┘

第一步：空看学习  →  第二步：产品通关  →  第三步：集攻研讨  →  第四步：带看签约
   ↓                   ↓                   ↓                   ↓
 踩盘踩盘             背背背              统一口径             签签签
```

## 二、空看学习（第一步）

### 2.1 选盘标准

| 类型 | 说明 |
|------|------|
| **责任盘上游项目** | 关注刚开盘项目，经历三个阶段：开盘火爆期→销售平稳期→降价清盘期 |
| **五环内/边项目** | 普遍火热、快销盘，客户主动咨询量大 |
| **降价力度大项目** | 同等价格面积变大、品质提升 |
| **区域附近新开盘** | 塔楼区域关注板楼/洋房，次新区域关注高端平层/别墅 |

### 2.2 空看前准备

| 准备项 | 内容 |
|--------|------|
| **时间安排** | 至少提前一天确定空看项目和时间 |
| **车辆安排** | 渠道经理/案场经理安排大巴车 |
| **学习标准** | 全程记笔记，新人可隐蔽录音；认真听讲；把自己当成真正买房的客户 |
| **携带材料** | 空看单页、笔 |

### 2.3 空看步骤

```
步骤一：大巴车到售楼处
   ↓
步骤二：了解在售产品信息
   ├─ 认真记笔记
   ├─ 区位图/沙盘认真记录
   └─ 准备好问题
   
步骤三：样板间
   ├─ 一定要录音（尤其精装项目）
   └─ 悄悄录

步骤四：总结
   ├─ 总结项目5-10个卖点
   ├─ 发店长编辑朋友圈文案
   └─ 区经评选3-5个优质模板，全区域转发
```

### 2.4 空看检核

| 检核项 | 执行人 |
|--------|--------|
| 当组总结5-10条卖点 | 店长 |
| 朋友圈3-5条优质模板 | 全区域转发 |
| 扫码答题 | 全员 |
| 组织产品通关/研讨会 | 区经 |

## 三、产品通关（第二步）

### 3.1 通关前准备

| 准备项 | 说明 |
|--------|------|
| **通关标准** | 10分钟内经纪人/店长能清晰完整讲解楼盘信息 |
| **准备材料** | 经纪人空看时记笔记（区位图、沙盘图打印出来） |
| **通关地点** | 优先选择售楼处空看完毕后就地开展；其次回区域以区域会形式 |

### 3.2 通关步骤

```
第一步：回售楼处（建议15分钟）
   ├─ 先研讨完善笔记
   ├─ 梳理行销思路
   └─ 每个伙伴都要开口讲

第二步：小组研讨（建议1小时）
   ├─ 区位：交通、医疗、商业、教育、板块解读
   ├─ 沙盘：户型产品解读
   └─ 总结：梳理5-10条卖点

第三步：行销考核（建议1小时）
   ├─ 店长向区经讲楼盘
   ├─ 经纪人向店长讲楼盘
   └─ 讲盘标准：板块解读+沙盘产品户型解读
```

### 3.3 通关常见问题

| 问题 | 原因/解决方案 |
|------|----------------|
| 张口讲不出楼盘 | 不了解项目，空看没学明白；不知道行销流程 |
| 不知道水平是否达标 | 区域现场研讨，总监/案场/渠道协助把关 |
| 不记笔记 | 意识先行，好记性不如烂笔头 |
| 整套流程太费时间 | 时间是值得的，专业是提升本领的第一步 |

## 四、集攻研讨会（第三步）

### 4.1 研讨会目标

> 约行程不是最终目的，更重要的是：
> - 提升自己的专业能力
> - 了解项目的受众群体
> - 主动有意识地盘活潜在客户
> - 专业自信地解答客户疑虑
> - 最后形成一组带看

### 4.2 研讨会流程

```
第一步：区经抽查
   ├─ 随机抽查10位经纪人
   ├─ 让自己整理的卖点念出来
   └─ 投屏话术，全区域一起读10遍

第二步：经纪人发朋友圈
   ├─ 编辑项目朋友圈模板
   ├─ 发至区域群
   └─ 店长整合5-10条发区经

第三步：评选最佳模板
   ├─ 区经汇总所有店长卖点
   └─ 评选3-5条最佳模板全员转发

第四步：邀约客户
   ├─ 从老客户、责任盘业主、租户、近期看房客户出发
   └─ 以火爆盘、优质好房、超高性价比等角度邀约
```

### 4.3 购买客户分类

| 客户类型 | 核心需求 |
|----------|----------|
| **首次置业** | 预算首付、小区品质、交通配套 |
| **置换改善** | 预算首付、面积置换、户型升级、特殊需求 |
| **投资** | 租户、升值潜力、地段位置、租金回报率 |
| **其他** | 出国在京留房、养老购房、居住环境 |

### 4.4 责任盘激活

| 业主类型 | 激活策略 |
|----------|----------|
| **自住业主** | 针对居住痛点做升级（塔楼换板楼、朝向不理想换南北通透等） |
| **租赁业主** | 将名下房产整合，换新换好、换出租回报率更高的 |
| **空置业主** | 代表有实力的客户群体，好项目好产品激发需求 |

### 4.5 常见问题应对

| 客户疑虑 | 标准应答 |
|----------|----------|
| 新房太远了不去 | 开车走高速20多分钟，享受居住面积增大、生活品质升级等 |
| 交房太晚不考虑 | 板块是国家新规划，交房时配套都起来，二手房价格一定高于一手 |
| 周边啥都没有 | 国家斥资打造，医疗商业教育配套已规划；可把配套标注地图发客户 |
| 要买直接找开发商 | 麦田全北京根据需求专业匹配，帮客观分析利弊 |

## 五、带看前中后（第四步）

### 5.1 带看前准备

#### 了解客户需求
| 核实项 | 内容 |
|--------|------|
| 落实客户需求 | 首付预算、置换改善、首套二套、投资养老、实际居住地 |
| 了解看房经历 | 看了多久房子、首选板块、中意房源、核心需求 |
| 了解购房资格 | 了解到婚姻状况 |

#### 同档次房源匹配
- 根据客户预算首付、板块位置，拿出房源价格表一一对应
- 做到"一带多看"提高客户转化率
- 先同一板块、同一价格段，再扩大范围

#### 带看前动作
| 动作 | 说明 |
|------|------|
| 麦脉报备 | 第一时间麦脉系统报备 |
| 案场微信报备 | 主看楼盘提前和案场沟通客户情况 |
| 打印材料 | 新房地图、房源价格表、带看一页纸 |
| 告知行程 | 告知客户今日行程安排 |

#### 带看前打印材料
| 材料 | 用途 |
|------|------|
| 新房地图 | 展示合作实力 |
| 房源价格表 | 随机应对匹配 |
| 带看一页纸 | 解答基础信息 |

### 5.2 带看中准备

| 步骤 | 内容 |
|------|------|
| **到售楼处** | 早于客户到达；保安询问登记；客户电话与报备一致 |
| **安排销售** | 前台登记；拿销售名片；扫码加销售和客户微信 |
| **销售接待中** | 全程认真陪同；夸奖销售专业敬业；找机会讲客户情况（尤其换房客户） |
| **看完后** | 主动邀约洽谈；配合销售聊出客户想法；无论在哪都不要谈其他项目 |

### 5.3 带看后准备

#### 带看后动作一
| 动作 | 说明 |
|------|------|
| 和客户一同离开 | 高佣金项目铺垫 |
| 针对客户需求再次匹配 | 必要的话PASS掉新房 |
| 第一时间拉沟通群 | 客户、店长、案场、销售、新房部总监 |
| 换房业主铺垫 | 不要对外说定了房，影响卖价 |

#### 带看后动作二：拉复访
| 数据 | 说明 |
|------|------|
| 带看新房复访率 | 仅12% |
| 复访转签约 | 4:1 |
| 结论 | 不是所有客户都能现场下定，需要不断对产品解读、疑虑解答 |

### 5.4 带看常见问题

| 问题 | 应对方案 |
|------|----------|
| 现场太忙客户无人接待 | 提前背项目知识；自己解答；现场逼定客户 |
| 销售接待太敷衍 | 找店长沟通；问是否身体不适；可联系案场换销售 |

## 六、新房交易流程

### 6.1 签约流程

```
第一步：认购
   ├─ 交5-100万小定
   └─ 付款成功开发商给收据证明

第二步：认购到草签
   ├─ 全款购买：1-2个月网签
   ├─ 贷款购买：2-3个月网签
   └─ 换房购买：4-5个月网签

第三步：草签
   └─ 补齐首付10%-40%

第四步：网签
   └─ 贷款客户与银行签合同
```

### 6.2 缴税交房

| 步骤 | 内容 |
|------|------|
| 第一步 | 网签合同完成、贷款银行审批通过、项目楼栋主体封顶，达到放款条件 |
| 第二步 | 期房项目交房：在买卖合同约定时间内且达到交付条件后统一领取钥匙 |
| 第三步 | 贷款客户开始还月供 |
| 第四步 | 开发商通知缴纳契税及公共维修金 |
| 第五步 | 办理房产证（一般合同约定交房日起两年内） |

**税费标准**：
- 契税：同二手房
- 公维：别墅50元/平；楼体≦6层100元/平；楼体≧6层200元/平

### 6.3 回款结佣

| 步骤 | 说明 |
|------|------|
| 常规周期 | 客户网签后3-6个月 |
| 经纪人 | 提供客户凭证 |
| 案场 | 整合提交结佣材料 |
| 开发商 | 审核发票材料、通知麦田财务开票 |
| 麦田财务 | 开票、案场整合提交、开发商财务审核 |
| 财务 | 次月10号发工资 |

## 七、新房业务规则

### 7.1 客户报备制度

```
经纪人麦脉系统报备
       ↓
案场微信报备
       ↓
案场30分钟内提交开发商审核
       ↓
有效/无效客户反馈经纪人
```

### 7.2 客户界定规则

| 类型 | 规则 |
|------|------|
| **内部界定** | 麦脉报备成功后保护2日产生带看；系统保护30日 |
| **开发商界定** | 客户已去售楼处或在保护期/永久保护则属于开发商客户 |

**内部界定案例**：
- 案例一：报备A项目且产生带看，保护30天
- 案例二：报备但未产生带看，2天后可被其他经纪人报备
- 案例三：同一客户只保护同一带看项目，其他楼盘可正常报备

### 7.3 红黄线制度

| 红线 | 行为 |
|------|------|
| **红线** | 拦截尾随客户获取信息；明知客户在麦田正面谈/签约，通过诋毁房源、谎报成交价、放佣金等方式搅单 |
| **黄线** | 明示暗示顾客更换信息再次报备；将客源推荐到其他公司签约；与开发商合作牟取私利；炒楼行为 |
| **黑线** | 购买一手房，自签订买卖合同第二日起不满1年（含）再次出售 |

## 八、总结：传统新房作业关键点

### 8.1 核心流程

| 阶段 | 关键词 | 产出 |
|------|--------|------|
| 空看学习 | 踩、记、摄 | 5-10条卖点、朋友圈模板 |
| 产品通关 | 背、讲、考 | 10分钟完整讲解楼盘 |
| 集攻研讨 | 统一口径、激活 | 最佳模板、全员转发 |
| 带看签约 | 配、陪、追 | 成交 |

### 8.2 经纪人能力要求

| 能力 | 说明 |
|------|------|
| **楼盘知识** | 10分钟完整讲解楼盘信息 |
| **区位解读** | 交通、医疗、商业、教育 |
| **产品解读** | 户型、面积、价格 |
| **异议处理** | 远、旧、配套差、价格高等常见问题 |
| **客户匹配** | 根据需求推荐合适楼盘 |

### 8.3 管理层职责

| 角色 | 职责 |
|------|------|
| **店长** | 整合卖点、协助把关、组织本店集攻会 |
| **区经** | 抽查考核、整合区域话术、组织区域集攻会 |
| **案场** | 提供物料（9宫格图、视频）、参与培训 |

### 8.4 传统流程痛点

| 痛点 | 说明 |
|------|------|
| **信息分散** | 每个经纪人自己整理，没有统一管理 |
| **质量不一** | 全靠个人能力，笔记、口径参差不齐 |
| **效率低下** | 踩盘靠腿，学习靠背，转化靠运气 |
| **反馈缺失** | 发出去的内容效果如何？不知道 |
| **能力难量化** | 产品力无法评估，考核靠感觉 |
| **物料难统一** | 每个盘要单独准备物料，重复劳动 |

## 九、与AI平台的能力对比

| 环节 | 传统方式 | AI平台赋能 |
|------|----------|------------|
| **空看学习** | 靠腿踩盘、记笔记 | AI自动采集+标准化物料库 |
| **产品通关** | 靠背书、靠人考 | AI训练+自动评分+实时反馈 |
| **内容生成** | 自己写朋友圈 | AI自动生成多平台文案 |
| **客户匹配** | 靠经验判断 | AI基于客户画像智能推荐 |
| **能力评估** | 靠感觉、无量化 | AI自动记录+能力雷达图 |
| **管理层** | 靠问、靠看、靠催 | 数据看板+实时追踪 |
