## 三、页面功能详情

### 3.1 内容日历页

#### 功能点

| # | 功能点 | 说明 | 交互 | 状态 |
|---|--------|------|------|------|
| 1 | 日历视图 | 按月展示日历 | 月份切换 | P0 |
| 2 | 今日标记 | 当天有任务时标记 | 红点标记 | P0 |
| 3 | 任务列表 | 显示当天所有任务 | 列表展示 | P0 |
| 4 | 任务卡片 | 显示任务基本信息 | 卡片样式 | P0 |
| 5 | 通知入口 | 右上角铃铛图标 | 按钮→弹窗 | P0 |
| 6 | 新建任务 | 创建新任务 | 按钮 | P2 |
| 7 | Tab切换 | 全部/探盘/话术/内容 | Tab切换 | P0 |

#### 任务类型

| 类型 | 图标 | 说明 |
|------|------|------|
| 探盘打卡 | 📍 | 探盘任务通知 |
| 话术训练 | 💬 | 话术训练任务 |
| 内容发布 | 📝 | 内容发布任务 |

#### 跳转关系

```
内容日历页
    ├── 点击通知铃铛 → 通知面板弹窗
    ├── 点击任务卡片 → 任务详情页
    ├── 切换Tab → 刷新任务列表
    └── 切换月份 → 刷新日历数据
```

---

### 3.2 通知面板(弹窗)

#### 功能点

| # | 功能点 | 说明 | 交互 | 状态 |
|---|--------|------|------|------|
| 1 | 全部通知 | 显示所有任务通知 | 列表 | P0 |
| 2 | 分类筛选 | 探盘/话术/内容 | Tab切换 | P0 |
| 3 | 通知列表 | 按时间倒序排列 | 列表 | P0 |
| 4 | 通知项 | 显示任务标题/时间/状态 | 只读 | P0 |
| 5 | 已读/未读 | 区分通知状态 | 样式区分 | P0 |
| 6 | 点击跳转 | 点击通知跳转详情 | 点击→详情页 | P0 |
| 7 | 关闭弹窗 | 关闭通知面板 | 按钮/遮罩 | P0 |

#### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| notificationId | string | 通知ID |
| type | enum | 探盘/话术/内容 |
| title | string | 通知标题 |
| content | string | 通知内容 |
| taskId | string | 关联任务ID |
| isRead | boolean | 是否已读 |
| createdAt | datetime | 发布时间 |

#### 跳转关系

```
通知面板弹窗
    └── 点击通知项 → 任务详情页 → 关闭弹窗
```

### 3.3 探盘任务详情页

#### 功能点

| # | 功能点 | 说明 | 交互 | 状态 |
|---|--------|------|------|------|
| 1 | 任务信息 | 显示楼盘/时间/要求 | 只读 | P0 |
| 2 | 四阶段进度 | 显示完成情况 | 进度条 | P0 |
| 3 | 执行入口 | 进入各阶段执行 | 按钮 | P0 |
| 4 | 话术训练入口 | 跳转训练页 | 按钮 | P0 |
| 5 | 提交任务 | 提交完成的任务 | 按钮 | P0 |

#### 跳转关系

```
探盘任务详情页
    ├── 阶段1定位打卡 → 打卡页
    ├── 阶段2点位打卡 → 点位页
    ├── 阶段3文案确认 → 文案页
    ├── 阶段4成果生成 → 成果页
    └── 话术训练 → speech-train.html
```

## 四、推送规则

### 4.1 推送时机

| 任务类型 | 推送时机 | 说明 |
|----------|----------|------|
| 探盘任务 | PC下发时 | 即时推送 |
| 话术训练 | 新房关联时 | 即时推送 |
| 内容发布 | 素材准备好时 | 定时推送 |

### 4.2 推送内容

| 类型 | 内容模板 |
|------|----------|
| 探盘任务 | 【探盘任务】您有一个新任务：{楼盘名称}，请及时处理 |
| 话术训练 | 【话术训练】{楼盘名称}的新训练项目已上线，快去练习吧 |
| 内容发布 | 【内容发布】今日内容发布任务已下发，请及时完成 |
