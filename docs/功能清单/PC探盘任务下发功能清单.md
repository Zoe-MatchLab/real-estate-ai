## 三、页面功能详情

### 3.1 探盘任务列表页

#### 功能点

| # | 功能点 | 说明 | 交互 | 状态 |
|---|--------|------|------|------|
| 1 | 任务筛选 | 按状态/楼盘/人员筛选 | 下拉选择 | P0 |
| 2 | 搜索任务 | 按任务名称搜索 | 输入搜索 | P1 |
| 3 | 任务列表 | 展示所有任务及状态 | 表格展示 | P0 |
| 4 | 批量下发 | 选中多条任务批量下发 | 复选框+按钮 | P0 |
| 5 | 批量导出 | 导出选中任务 | 复选框+按钮 | P1 |
| 6 | 批量修改人员 | 修改选中任务的执行人员 | 复选框+按钮 | P1 |
| 7 | 新建任务 | 创建新任务 | 按钮→表单页 | P0 |
| 8 | 查看详情 | 查看任务详情 | 点击→详情页 | P0 |

#### 任务状态

| 状态 | 说明 |
|------|------|
| 待分发 | 任务已创建，尚未下发 |
| 进行中 | 任务已下发，经纪人执行中 |
| 已提交 | 经纪人已提交，等待审核 |
| 已完成 | 审核通过，任务结束 |
| 已驳回 | 审核驳回，需重新执行 |
| 已逾期 | 超过截止时间未完成 |

#### 跳转关系

```
探盘任务列表页
    ├── 新建任务 → 新建任务表单页
    ├── 查看详情 → 任务详情页
    ├── 批量下发 → 确认弹窗 → 下发成功
    └── 筛选 → 刷新列表
```

---

### 3.2 新建/编辑任务表单页

#### 功能点

| # | 功能点 | 说明 | 交互 | 状态 |
|---|--------|------|------|------|
| 1 | 选择楼盘 | 选择执行探盘的楼盘 | 下拉选择 | P0 |
| 2 | 填写任务名称 | 自定义任务名称 | 输入框 | P0 |
| 3 | 选择执行人员 | 指定任务执行人 | 人员选择器 | P0 |
| 4 | 选择通知人员 | 任务通知接收人 | 人员选择器 | P1 |
| 5 | 设置截止时间 | 任务完成截止时间 | 日期选择器 | P0 |
| 6 | 设置优先级 | 高/中/低优先级 | 单选 | P1 |
| 7 | 任务要求 | 填写探盘要求 | 多行文本 | P1 |
| 8 | 保存草稿 | 保存为草稿不下发 | 按钮 | P1 |
| 9 | 确认下发 | 保存并下发任务 | 按钮→下发 | P0 |

#### 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| buildingId | string | 是 | 楼盘ID |
| taskName | string | 是 | 任务名称 |
| executorIds | array | 是 | 执行人员ID列表 |
| notifierIds | array | 否 | 通知人员ID列表 |
| deadline | datetime | 是 | 截止时间 |
| priority | enum | 否 | 高/中/低 |
| requirements | string | 否 | 任务要求 |

#### 跳转关系

```
新建任务表单页
    ├── 保存草稿 → 返回列表
    ├── 确认下发 → 下发成功 → 返回列表
    └── 取消 → 确认弹窗 → 返回列表
```

### 3.3 任务详情页

#### 功能点

| # | 功能点 | 说明 | 交互 | 状态 |
|---|--------|------|------|------|
| 1 | 任务信息 | 显示任务基本信息 | 只读 | P0 |
| 2 | 执行人员 | 显示执行人员列表 | 只读 | P0 |
| 3 | 执行进度 | 显示4阶段完成情况 | 进度条 | P0 |
| 4 | 提交内容 | 显示经纪人提交的内容 | 预览 | P0 |
| 5 | 审核操作 | 通过/驳回提交内容 | 按钮 | P0 |
| 6 | 驳回原因 | 填写驳回原因 | 输入框 | P0 |
| 7 | 审核记录 | 显示审核历史 | 只读 | P1 |
| 8 | 编辑任务 | 修改任务信息 | 按钮→编辑页 | P1 |
| 9 | 重新下发 | 重新下发任务 | 按钮 | P1 |
| 10 | 删除任务 | 删除任务 | 确认弹窗 | P1 |

#### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| taskId | string | 任务ID |
| taskName | string | 任务名称 |
| buildingName | string | 楼盘名称 |
| status | enum | 当前状态 |
| executorName | string | 执行人员姓名 |
| progress | object | 四阶段进度 |
| submittedContent | object | 提交内容 |
| reviewStatus | enum | 审核状态 |
| reviewComment | string | 审核意见 |

#### 跳转关系

```
任务详情页
    ├── 审核通过 → 状态更新 → 返回列表
    ├── 审核驳回 → 填写原因 → 返回列表
    ├── 编辑 → 编辑表单页
    └── 返回列表
```
