## 三、页面功能详情

### 3.1 训练项目选择页

#### 功能点

| # | 功能点 | 说明 | 交互 | 状态 |
|---|--------|------|------|------|
| 1 | 项目列表 | 展示可训练的楼盘项目 | 默认显示/下拉刷新 | P0 |
| 2 | 搜索项目 | 按项目名称搜索 | 输入搜索 | P1 |
| 3 | 训练状态 | 显示「待训练/已完成」 | 只读 | P0 |
| 4 | 完成记录 | 显示完成次数和评分 | 只读 | P1 |
| 5 | 开始训练按钮 | 点击进入AI对话 | 点击→训练页 | P0 |

#### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| projectName | string | 项目名称 |
| projectCover | string | 项目封面图 |
| status | enum | 待训练/已完成 |
| completedCount | number | 完成次数 |
| latestScore | number | 最新得分 |

#### 跳转关系

```
训练项目选择页
    └── 点击项目 → AI训练对话页
```

---

### 3.2 AI训练对话页

#### 功能点

| # | 功能点 | 说明 | 交互 | 状态 |
|---|--------|------|------|------|
| 1 | AI对话 | AI扮演客户进行对话 | 自动/手动发送 | P0 |
| 2 | 话术建议 | 显示推荐话术 | 气泡展示 | P1 |
| 3 | 评分展示 | 实时显示对话评分 | 实时更新 | P0 |
| 4 | 训练完成 | 显示训练结果和建议 | 结果页 | P0 |
| 5 | 退出训练 | 放弃当前训练 | 确认弹窗 | P1 |

#### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| messages | array | 对话消息列表 |
| score | number | 实时评分 0-100 |
| suggestions | array | 话术建议列表 |
| result | object | 训练结果 |

#### 跳转关系

```
AI训练对话页
    └── 训练完成 → 结果页 → 返回列表
```
