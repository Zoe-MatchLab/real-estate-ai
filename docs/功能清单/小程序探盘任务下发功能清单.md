## 三、页面功能详情

### 3.1 任务列表页

#### 功能点

| # | 功能点 | 说明 | 交互 | 状态 |
|---|--------|------|------|------|
| 1 | 任务筛选 | 按状态筛选 | 下拉选择 | P0 |
| 2 | 任务搜索 | 按楼盘名称搜索 | 输入搜索 | P1 |
| 3 | 任务列表 | 展示所有任务 | 列表展示 | P0 |
| 4 | 任务卡片 | 显示任务基本信息 | 卡片样式 | P0 |
| 5 | 状态标签 | 显示任务当前状态 | 标签样式 | P0 |
| 6 | 新建任务 | 创建新任务 | 按钮→表单页 | P0 |
| 7 | 查看详情 | 查看任务详情 | 点击→详情页 | P0 |

#### 任务状态

| 状态 | 说明 |
|------|------|
| 待分发 | 任务已创建 |
| 进行中 | 执行中 |
| 已提交 | 已提交审核 |
| 已完成 | 审核通过 |
| 已驳回 | 审核驳回 |
| 已逾期 | 超时未完成 |

#### 跳转关系

```
任务列表页
    ├── 新建任务 → 新建任务表单页
    ├── 查看详情 → 任务详情页(task-detail.html)
    └── 筛选 → 刷新列表
```

---

### 3.2 新建/编辑任务表单页

#### 功能点

| # | 功能点 | 说明 | 交互 | 状态 |
|---|--------|------|------|------|
| 1 | 选择楼盘 | 选择执行探盘的楼盘 | 下拉选择 | P0 |
| 2 | 任务名称 | 自定义或自动生成 | 输入/自动 | P0 |
| 3 | 执行人员 | 指定执行人 | 人员选择器 | P0 |
| 4 | 截止时间 | 设置完成时间 | 日期选择 | P0 |
| 5 | 任务要求 | 填写探盘要求 | 多行文本 | P1 |
| 6 | 保存下发 | 保存并下发任务 | 按钮 | P0 |
| 7 | 保存草稿 | 仅保存不下发 | 按钮 | P1 |
| 8 | 取消返回 | 取消并返回列表 | 按钮 | P0 |

#### 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| buildingId | string | 是 | 楼盘ID |
| buildingName | string | 只读 | 楼盘名称 |
| taskName | string | 是 | 任务名称 |
| executorId | string | 是 | 执行人员ID |
| deadline | datetime | 是 | 截止时间 |
| requirements | string | 否 | 任务要求 |

#### 跳转关系

```
新建/编辑任务表单页
    ├── 保存下发 → 下发成功 → 返回列表
    ├── 保存草稿 → 保存成功 → 返回列表
    └── 取消 → 返回列表
```

## 四、与PC端区别

| 功能 | PC端 | 小程序端 |
|------|------|----------|
| 批量下发 | ✅ 支持 | ❌ 不支持 |
| 批量导出 | ✅ 支持 | ❌ 不支持 |
| 批量修改人员 | ✅ 支持 | ❌ 不支持 |
| 单条任务配置 | ✅ 支持 | ✅ 支持 |
| 任务审核 | ✅ 支持 | ❌ 不支持（小程序仅执行） |

## 五、权限控制

| 角色 | 可查看 | 可操作 |
|------|--------|--------|
| 经纪人 | 自己的任务 | 查看/执行 |
| 运营人员 | 全部任务 | 创建/编辑/下发 |
| 业务总监 | 团队任务 | 查看 |
| 管理员 | 全部任务 | 全权限 |
