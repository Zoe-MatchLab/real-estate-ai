## 一、页面概述

### 1.1 基本信息

| 属性 | 值 |
|------|------|
| 页面名称 | 个人中心 |
| 路由 | /profile |
| 优先级 | P0 |
| 所属端 | C端小程序/H5 |

### 1.2 用户角色

| 角色 | 说明 |
|------|------|
| 经纪人 | 主要使用者，查看个人信息、业绩数据、使用功能入口 |
| 管理人员 | 可查看团队成员业绩数据（后续规划） |

### 1.3 核心目标

1. 展示经纪人个人信息（头像、姓名、职位、积分、排名）
2. 展示个人业绩数据（本月业绩、今日概览）
3. 提供快捷功能入口（探盘任务、话术训练、我的物料库等）
4. 提供通用工具入口（客户管理、帮助中心等）
5. 提供账号设置入口

## 三、功能模块详解

### 3.1 顶部个人信息区

#### 模块说明
展示当前登录用户的基本信息

#### 功能点

| # | 功能点 | 说明 | 交互 | 状态 |
|---|--------|------|------|------|
| 1 | 显示头像 | 圆形头像，默认显示姓名首字母 | 点击查看大图 | 静态 |
| 2 | 显示姓名 | 用户真实姓名 | - | 静态 |
| 3 | 显示职位标签 | 如"高级置业顾问" | - | 静态 |
| 4 | 显示区域和工号 | 格式：所属区域：XX区 \| 工号：XXX | - | 静态 |
| 5 | 显示积分 | 积分图标+数值 | - | 静态 |
| 6 | 显示排名 | 排名图标+数值 | - | 静态 |
| 7 | 设置按钮 | 跳转设置页面 | 点击→设置弹窗/页面 | 可交互 |
| 8 | 查看名片按钮 | 查看个人电子名片 | 点击→名片页 | 可交互 |

#### 字段说明

| 字段 | 类型 | 示例 |
|------|------|------|
| name | string | 张三 |
| avatar | string(url) | https://xxx.com/avatar.jpg |
| title | string | 高级置业顾问 |
| area | string | 海淀区 |
| employeeId | string | BJ00123 |
| points | number | 2860 |
| rank | number | 6 |

## 五、状态说明

### 5.1 页面状态

| 状态 | 说明 | 展示内容 |
|------|------|----------|
| 加载中 | 获取用户数据中 | Skeleton骨架屏 |
| 加载成功 | 数据获取完成 | 完整页面内容 |
| 加载失败 | 网络错误/服务器异常 | 错误提示+重试按钮 |

### 5.2 徽章状态

| 类型 | 颜色 | 含义 |
|------|------|------|
| 红色徽章 | bg-red-50 text-red-500 | 待办、紧急 |
| 橙色徽章 | bg-orange-50 text-orange-500 | 进行中、待完成 |
| 灰色徽章 | bg-gray-100 text-gray-500 | 统计数量 |
