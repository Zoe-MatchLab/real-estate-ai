## 三、页面功能详情

### 3.1 AI工具首页

#### 功能点

| # | 功能点 | 说明 | 交互 | 状态 |
|---|--------|------|------|------|
| 1 | 快捷入口 | 显示主要功能入口 | 图标网格 | P0 |
| 2 | 探盘任务入口 | 跳转任务列表 | 点击→日历页 | P0 |
| 3 | 话术训练入口 | 跳转训练页 | 点击→训练页 | P0 |
| 4 | 内容发布入口 | 跳转发布页 | 点击→发布页 | P0 |
| 5 | 最近任务 | 显示最近执行的任务 | 列表 | P1 |
| 6 | 通知铃铛 | 显示未读通知数 | 点击→通知面板 | P0 |

#### 跳转关系

```
AI工具首页
    ├── 探盘任务 → calendar.html
    ├── 话术训练 → speech-train.html
    ├── 内容发布 → (暂未开放提示)
    └── 通知铃铛 → 通知面板弹窗
```

### 3.2 日历页面(任务列表)

#### 功能点

| # | 功能点 | 说明 | 交互 | 状态 |
|---|--------|------|------|------|
| 1 | 日历视图 | 月份日历展示 | 滑动切换 | P0 |
| 2 | 任务标记 | 当天有任务时显示红点 | 只读 | P0 |
| 3 | 任务列表 | 当天任务卡片列表 | 列表展示 | P0 |
| 4 | 任务卡片 | 任务基本信息+状态 | 卡片样式 | P0 |
| 5 | 通知入口 | 右上角铃铛 | 点击→弹窗 | P0 |
| 6 | 分类Tab | 全部/探盘/话术/内容 | Tab切换 | P0 |
| 7 | 下拉刷新 | 刷新任务列表 | 下拉操作 | P1 |

#### 跳转关系

```
日历页面
    ├── 点击通知铃铛 → 通知面板弹窗
    ├── 点击任务卡片 → 任务详情页
    └── 切换Tab → 刷新列表
```

### 3.3 通知面板弹窗

#### 功能点

| # | 功能点 | 说明 | 交互 | 状态 |
|---|--------|------|------|------|
| 1 | 全部通知 | 显示所有通知 | 列表 | P0 |
| 2 | 分类通知 | 按类型筛选 | Tab切换 | P0 |


### 3.4 探盘任务详情页

#### 功能点

| # | 功能点 | 说明 | 交互 | 状态 |
|---|--------|------|------|------|
| 1 | 任务信息 | 楼盘名称/地址/时间 | 只读 | P0 |
| 2 | 四阶段执行 | 4阶段分步引导 | 阶段卡片 | P0 |
| 3 | 阶段1：定位打卡 | 到场定位 | 地图+打卡 | P0 |
| 4 | 阶段2：点位打卡 | 各个点位采集 | 拍照/视频 | P0 |
| 5 | 阶段3：文案确认 | 确认生成文案 | 预览+确认 | P0 |
| 6 | 阶段4：成果生成 | 查看所有成果 | 成果展示 | P0 |
| 7 | 话术训练 | 跳转训练页 | 按钮 | P0 |
| 8 | 提交任务 | 提交审核 | 按钮 | P0 |

#### 跳转关系

```
探盘任务详情页
    ├── 阶段1 → 定位打卡页 → 返回
    ├── 阶段2 → 点位打卡页 → 返回
    ├── 阶段3 → 文案确认页 → 返回
    ├── 阶段4 → 成果生成页 → 返回
    └── 话术训练 → speech-train.html
```

## 五、数据结构

### 5.1 任务通知

| 字段 | 类型 | 说明 |
|------|------|------|
| notificationId | string | 通知ID |
| type | enum | 探盘/话术/内容 |
| title | string | 通知标题 |
| content | string | 通知内容 |
| taskId | string | 关联任务ID |
| isRead | boolean | 是否已读 |
| createdAt | datetime | 创建时间 |

### 5.2 任务卡片

| 字段 | 类型 | 说明 |
|------|------|------|
| taskId | string | 任务ID |
| type | enum | 探盘/话术/内容 |
| title | string | 任务标题 |
| buildingName | string | 楼盘名称 |
| status | enum | 状态 |
| deadline | datetime | 截止时间 |
| progress | object | 进度信息 |
