## 三、PC端功能详情

### 3.1 新盘列表页

#### 功能点

| # | 功能点 | 说明 | 交互 | 状态 |
|---|--------|------|------|------|
| 1 | 楼盘列表 | 分页展示所有楼盘 | 表格展示 | P0 |
| 2 | 状态筛选 | 按状态筛选楼盘 | 下拉选择 | P0 |
| 3 | 搜索楼盘 | 按名称/区域搜索 | 输入搜索 | P1 |
| 4 | 新建楼盘 | 创建新楼盘 | 按钮→表单页 | P0 |
| 5 | 审核操作 | 通过/驳回楼盘 | 按钮 | P0 |
| 6 | 编辑楼盘 | 修改楼盘信息 | 按钮→编辑页 | P0 |
| 7 | 上下架 | 上架/下架楼盘 | 开关 | P1 |
| 8 | 导出数据 | 导出楼盘列表 | 按钮 | P2 |

#### 楼盘状态

| 状态 | 说明 |
|------|------|
| 待提交 | 草稿状态，未提交审核 |
| 审核中 | 已提交，等待审核 |
| 已通过 | 审核通过，已上线 |
| 已驳回 | 审核驳回，需修改 |
| 已下架 | 手动下架 |

#### 跳转关系

```
新盘列表页
    ├── 新建楼盘 → 新盘表单页
    ├── 查看详情 → 楼盘详情页
    ├── 审核通过 → 状态更新
    ├── 审核驳回 → 填写原因 → 状态更新
    └── 编辑 → 新盘编辑页
```

---

### 3.2 新盘创建/编辑表单页

#### 功能点

| # | 功能点 | 说明 | 交互 | 状态 |
|---|--------|------|------|------|
| 1 | 基本信息 | 楼盘名称/区域/地址等 | 输入框 | P0 |
| 2 | 价格信息 | 均价/总价/单价 | 输入框 | P0 |
| 3 | 户型信息 | 户型图/面积/居室数 | 上传+输入 | P0 |
| 4 | 详细信息 | 开发商/容积率/绿化率等 | 输入框 | P1 |
| 5 | 楼盘封面 | 上传封面图 | 图片上传 | P0 |
| 6 | 户型图册 | 上传多张户型图 | 图片上传 | P0 |
| 7 | 视频资料 | 上传楼盘视频 | 视频上传 | P1 |
| 8 | 保存草稿 | 保存不提交审核 | 按钮 | P1 |
| 9 | 提交审核 | 提交待审核 | 按钮 | P0 |

#### 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| name | string | 是 | 楼盘名称 |
| region | object | 是 | 区域信息 |
| address | string | 是 | 详细地址 |
| avgPrice | number | 是 | 均价 |
| coverImage | string | 是 | 封面图 |
| houseTypes | array | 否 | 户型列表 |
| developer | string | 否 | 开发商 |
| videoUrl | string | 否 | 视频链接 |

#### 跳转关系

```
新盘表单页
    ├── 保存草稿 → 返回列表
    ├── 提交审核 → 提交成功 → 返回列表
    └── 取消 → 确认弹窗 → 返回列表
```
