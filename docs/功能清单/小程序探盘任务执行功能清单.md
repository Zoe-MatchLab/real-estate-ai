## 三、页面功能详情

### 3.1 任务详情页

#### 功能点

| # | 功能点 | 说明 | 交互 | 状态 |
|---|--------|------|------|------|
| 1 | 任务信息 | 显示楼盘名称/地址/要求 | 只读 | P0 |
| 2 | 执行人员 | 显示执行人员信息 | 只读 | P0 |
| 3 | 截止时间 | 显示任务截止时间 | 只读 | P0 |
| 4 | 四阶段进度 | 显示4阶段完成情况 | 进度条 | P0 |
| 5 | 阶段1入口 | 定位打卡 | 按钮→打卡页 | P0 |
| 6 | 阶段2入口 | 点位打卡 | 按钮→点位页 | P0 |
| 7 | 阶段3入口 | 文案确认 | 按钮→文案页 | P0 |
| 8 | 阶段4入口 | 成果生成 | 按钮→成果页 | P0 |
| 9 | 话术训练入口 | 进入产品力训练 | 按钮 | P0 |
| 10 | 提交任务 | 提交完成的任务 | 按钮 | P0 |

#### 任务状态

| 状态 | 说明 |
|------|------|
| 待执行 | 收到任务，尚未开始 |
| 进行中 | 正在执行中 |
| 已提交 | 已提交，等待审核 |
| 已完成 | 审核通过 |
| 已驳回 | 审核驳回，需修改 |
| 已逾期 | 超过截止时间 |

#### 跳转关系

```
任务详情页
    ├── 阶段1定位打卡 → 定位打卡页 → 返回详情
    ├── 阶段2点位打卡 → 点位打卡页 → 返回详情
    ├── 阶段3文案确认 → 文案确认页 → 返回详情
    ├── 阶段4成果生成 → 成果生成页 → 返回详情
    └── 话术训练 → speech-train.html
```

## 四、字段说明

### 4.1 任务信息

| 字段 | 类型 | 说明 |
|------|------|------|
| taskId | string | 任务ID |
| taskName | string | 任务名称 |
| buildingName | string | 楼盘名称 |
| buildingAddress | string | 楼盘地址 |
| executorName | string | 执行人员 |
| deadline | datetime | 截止时间 |
| status | enum | 任务状态 |
| progress | object | 四阶段进度 |

### 4.2 打卡信息

| 字段 | 类型 | 说明 |
|------|------|------|
| location | object | 经纬度坐标 |
| accuracy | number | 定位精度 |
| checkInTime | datetime | 打卡时间 |
| photos | array | 打卡照片 |
| videos | array | 打卡视频 |

### 4.3 成果信息

| 字段 | 类型 | 说明 |
|------|------|------|
| photos | array | 照片列表 |
| videos | array | 视频列表 |
| audios | array | 录音列表 |
| copywriter | string | 最终文案 |
| submittedAt | datetime | 提交时间 |
