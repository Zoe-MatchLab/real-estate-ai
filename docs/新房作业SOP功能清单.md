## 二、SOP流程总览

```
┌─────────────────────────────────────────────────────────────┐
│                    新房作业SOP流程                          │
└─────────────────────────────────────────────────────────────┘

阶段1：新盘上线
    PC: 新盘管理 → 楼盘录入 → 提交审核
                    ↓
    PC: 审核人员 → 审核通过 → 楼盘上线
                    ↓
    系统: 自动关联训练项目

阶段2：任务下发
    PC: 探盘任务 → 创建任务 → 选择楼盘 → 配置人员 → 下发
                    ↓
    小程序: 经纪人收到推送通知
                    ↓
    PC: 审核探盘结果 → 通过/驳回

阶段3：能力训练
    小程序: 经纪人进行话术训练
                    ↓
    系统: 更新产品力评分

阶段4：内容发布
    小程序: 基于探盘素材生成内容
                    ↓
    小程序: 发布到各平台(小红书/公众号/视频号/朋友圈)

阶段5：业绩管理
    小程序: 业绩数据自动统计
                    ↓
    PC: 管理看板查看数据
```

## 五、数据流转

### 5.1 新盘→探盘任务

```
新盘审核通过
    ↓
系统自动生成探盘任务(草稿)
    ↓
运营人员编辑并下发
    ↓
经纪人接收任务
```

### 5.2 新盘→话术训练

```
新盘审核通过
    ↓
系统自动关联训练项目
    ↓
经纪人可在话术训练中看到
```

### 5.3 探盘任务→物料库

```
探盘任务完成
    ↓
成果(照片/视频/文案)自动存入物料库
    ↓
经纪人可查看/使用
```

### 5.4 物料库→内容发布

```
选择物料库内容
    ↓
AI生成文案/视频
    ↓
发布到各平台
    ↓
业绩数据统计
```
