## 一、产品概述

### 1.1 页面定位

个人中心是小程序的**用户主页**，承载用户基本信息展示、作业数据看板、会员状态、功能入口和底部导航等功能。

### 1.2 核心功能

| 模块 | 说明 | 状态 |
|------|------|------|
| 顶部导航 | 返回上一页、设置入口 | ✅ 已实现 |
| 用户信息 | 头像、昵称、经纪人标签 | ✅ 已实现 |
| **今日作业概览** | 任务数据看板（新增） | 🆕 待实现 |
| 会员兑换入口 | 跳转会员兑换页面 | ✅ 已实现 |
| 会员卡片 | 显示会员等级、到期时间、续费入口 | ✅ 已实现 |
| 功能入口 | 活动中心、算力、邀请、训练记录、收藏、足迹、付费记录、设置 | ✅ 已实现 |
| 底部TabBar | 5项导航：首页/功能/助手/日历/我的 | ✅ 已实现 |

#### 3.3.1 卡片布局

```
┌─────────────────────────────────────────┐
│  📋 今日作业概览            查看详情 >  │
├─────────────────────────────────────────┤
│                                         │
│  ┌────────┐ ┌────────┐ ┌────────┐     │
│  │   5    │ │  3/5   │ │  60%   │     │

│  └────────┘ └────────┘ └────────┘     │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ ⭐ 话术评分  85 分     ↑+5     │   │
│  │ ██████████░░░░░░░░░░░░░░░░░  │   │
│  │ 较上周提升                       │   │

│  ⚠️ 2 条数据异常待处理                 │
│                                         │
└─────────────────────────────────────────┘
```

#### 3.3.2 数据指标定义

| 指标 | 计算方式 | 数据类型 | 说明 |
|------|----------|----------|------|
| **总任务数** | 今日所有任务之和 | 整数 | 含探盘任务、话术训练等 |
| **已完成数** | 状态=已完成的任务数/总任务数 | 分数 | 格式：已完成/总数 |
| **完成率** | 已完成数/总任务数×100% | 百分比 | 整数，不带%符号 |
| **话术评分** | 最近30天训练平均分 | 0-100整数 | 最近一次评分作为趋势基准 |
| **评分趋势** | 本期评分-上期评分 | 整数 | 正数↑+N，负数↓-N |
| **数据异常** | 待处理异常数量 | 整数 | 为0时隐藏此行 |

#### 3.3.3 样式规范

**卡片整体**：

| 属性 | 值 |
|------|---|
| 背景 | 白色 #fff |
| 圆角 | 12px |
| 内边距 | 16px |
| 外边距 | 15px |
| 顶部外边距 | 0（紧贴用户信息） |

**数据指标项**：

| 属性 | 值 |
|------|---|
| 布局 | flex，三个均分 |
| 数字大小 | 24px |
| 数字字重 | 700（加粗） |
| 数字颜色 | #1D2129 |
| 标签大小 | 12px |
| 标签颜色 | #86909C |
| 间距 | 数字与标签8px |

**话术评分行**：

| 属性 | 值 |
|------|---|
| 布局 | 整行，flex |
| 评分数字 | 18px，加粗，橙色 #FA8C16 |
| 趋势标签 | 绿色 #00B42A 或红色 #FA5151 |
| 进度条高度 | 6px |
| 进度条圆角 | 3px |
| 进度条背景 | #F2F3F5 |
| 进度条填充 | 橙色渐变 |

**异常提示行**：

| 属性 | 值 |
|------|---|
| 图标 | ⚠️ 或 i 图标 |
| 文字颜色 | #FA5151（红色） |
| 字体大小 | 13px |
| 点击行为 | 跳转异常处理页（暂无页面时提示） |

#### 3.3.4 交互说明

| 元素 | 交互 | 说明 |
|------|------|------|
| 卡片标题「今日作业概览」 | 无交互 | 静态展示 |
| 「查看详情」 | 点击跳转 | 跳转任务详情页 task-detail.html |
| 话术评分进度条 | 点击 | 跳转话术训练页 speech-train.html |
| 异常提示行 | 点击 | 提示「功能即将上线」或跳转处理页 |
| 指标数字 | 无交互 | 静态展示 |

#### 3.3.5 状态处理

| 数据状态 | 显示内容 |
|----------|----------|
| 有任务 | 正常显示各项指标 |
| 无任务（空状态） | 卡片显示「今日暂无任务安排」 |
| 数据加载中 | 显示骨架屏或占位 |
| 数据异常 | 指标显示「--」 |
| 话术评分<60分 | 显示红色警告样式 |
| 话术评分60-80分 | 显示橙色正常样式 |
| 话术评分>80分 | 显示绿色优秀样式 |
| 异常数=0 | 隐藏异常提示行 |

```javascript

  todayOverview: {

    // 数据异常
    dataAlerts: {
      count: 2,  // 异常数量，为0时隐藏此行
```
