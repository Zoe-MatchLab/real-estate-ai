## 一、产品概述

### 1.1 页面定位

个人中心是小程序的**用户主页**，承载用户基本信息展示、会员状态、功能入口和底部导航等功能。

### 1.2 核心功能

| 模块 | 说明 | 状态 |
|------|------|------|
| 顶部导航 | 返回上一页、设置入口 | ✅ 已实现 |
| 用户信息 | 头像、昵称、经纪人标签 | ✅ 已实现 |
| 会员兑换入口 | 跳转会员兑换页面 | ✅ 已实现 |
| 会员卡片 | 显示会员等级、到期时间、续费入口 | ✅ 已实现 |
| 功能入口 | 活动中心、算力、邀请、训练记录、收藏、足迹、付费记录、设置 | ✅ 已实现 |
| 底部TabBar | 5项导航：首页/功能/助手/日历/我的 | ✅ 已实现 |

## 三、功能详解

### 3.1 顶部导航

**样式**：
- 高度：44px（含状态栏）
- 背景：与页面顶部渐变背景融合
- 左侧返回按钮：32px圆形区域
- 中间标题：17px加粗
- 右侧更多按钮：32px圆形区域

**交互**：

| 元素 | 交互 | 跳转页面 |
|------|------|----------|
| 返回按钮 `←` | 点击返回上一页 | history.back() |
| 标题 | 无交互 | - |
| 更多按钮 `•••` | 点击进入设置页 | settings.html |

### 3.2 用户信息区域

**布局**：
```
┌─────────────────────────────────────────┐
│  [头像]  陈佳佳 [经纪人]  [会员兑换]    │
└─────────────────────────────────────────┘
```

**样式**：

| 元素 | 样式 | 说明 |
|------|------|------|
| 头像 | 60px圆形，白色边框2px，阴影 | 可查看大图 |
| 昵称 | 18px加粗 | 显示用户昵称 |
| 经纪人标签 | 橙色渐变胶囊 | 表示认证身份 |
| 会员兑换按钮 | 橙色渐变胶囊 | 跳转会员页 |

**交互**：

| 元素 | 交互 | 跳转页面 |
|------|------|----------|
| 头像 | 点击查看大图 | - |
| 会员兑换按钮 | 点击 | vip.html |

### 3.3 会员卡片

## 五、数据结构

### 5.1 用户信息

```javascript
{
  userId: "USER_xxx",
  nickname: "陈佳佳",
  avatar: "https://xxx.com/avatar.jpg",
  role: "agent",  // agent(经纪人)/manager(管理员)
  
  // 会员信息
  member: {
    level: 1,  // 0=无会员, 1=白银, 2=黄金, 3=钻石
    levelName: "白银会员",
    expireTime: "2026-04-23T00:00:00Z",  // 15天后到期
    daysLeft: 15
  }
}
```
