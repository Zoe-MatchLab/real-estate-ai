## 一、产品概述

### 1.1 页面定位

个人中心是小程序的**作业管理中心**，承载用户信息展示、任务入口、业绩数据、个人成长、物料管理等功能，是新房作业流程的经纪人个人数据聚合页面。

### 1.2 核心功能（v2.0新增）

| 功能模块 | 说明 | 关联SOP |
|----------|------|----------|
| 个人信息 | 头像、昵称、经纪人认证标签 | - |
| 今日作业动态 | 探盘任务/话术训练/数据异常提醒 | 第二/三阶段 |
| 我的业绩 | 当月作业数据（探盘/发布/线索/报备/带看/成交） | 管理层看板 |
| 产品力成长 | 等级/能力分/下一级条件 | 第三阶段 |
| 我的物料库 | 探盘成果（照片/视频/文案）管理 | 第二阶段 |
| 管理层数据看板 | 经纪人视角数据（各平台发布/投流/咨询） | 管理层看板 |
| 功能入口 | 算力、收藏、活动、邀请、足迹、设置 | - |
| 会员体系 | 会员等级、权益展示（即将上线） | - |

### 3.1 个人信息区域

#### 3.1.1 布局说明

| 元素 | 样式 | 说明 |
|------|------|------|
| 头像 | 圆形，60px，带白色边框 | 点击可查看大图或更换头像 |
| 昵称 | 16px，加粗 | 显示用户昵称 |

| 交互 | 行为 |
|------|------|

| 点击编辑按钮 | 跳转至「编辑资料」页面 |
| 长按头像/昵称 | 弹出操作菜单（分享/复制） |

#### 3.1.3 状态说明

| 状态 | 显示内容 |
|------|----------|
| 已登录 | 正常显示头像和昵称 |
| 未登录 | 显示默认头像和「点击登录」 |
| 经纪人已认证 | 昵称后显示认证标签 |

#### 3.3.1 第一组菜单

| 属性 | 值 |
|------|---|
| 图标 | 📅 日历图标（橙色背景） |
| 名称 | 活动中心 |

| 属性 | 值 |
|------|---|

| 名称 | 算力 |
| 状态 | 正常 |

**算力说明**：
- 算力是用户参与AI功能（话术训练、内容生成等）的消耗资源
- 用户可通过完成任务、每日签到等方式获取算力

| 属性 | 值 |
|------|---|
| 图标 | 📁 文件袋图标（橙色背景） |
| 名称 | 收藏 |
| 内容 | X个收藏 |
| 状态 | 正常 |
| 入口 | 跳转至「我的收藏」页面 |

**收藏内容**：
- 用户收藏的楼盘信息
- 用户收藏的文案/视频模板
- 用户收藏的学习内容
