## 一、项目概述

### 1.1 项目背景

随着新房市场竞争加剧，传统经纪人作业模式面临四大核心挑战：

| 挑战 | 现状痛点 |
|------|---------|
| **信息获取滞后** | 经纪人依赖线下踩盘，无法快速全面了解楼盘 |
| **能力参差不齐** | 产品知识掌握程度不一，约看率、成交率不稳定 |
| **物料质量不一** | 探盘成果标准不统一，新媒体内容质量难以保障 |
| **管理抓手缺失** | 管理层难以实时掌握任务执行进度，缺乏数据支撑 |

### 1.2 产品定位

**新房新媒体获客作业标准定义者 + 能力中台**

本质：把经纪人作业能力从"个体经验"升级为"系统化输出"，解决规模化发展的核心瓶颈。

### 1.3 核心Slogan

> **「探得真材，炼出专业，推向市场，约见成交」**

## 三、业务流程（探-练-推-约）

### 3.1 业务流程图

```
                    ┌──────────────────────────────┐
                    │      【探】探盘任务           │
                    │  区域经理下发 → 经纪人领取    │
                    │  GPS签到 → 采集楼盘信息       │
                    │  拍摄照片/视频 → 提交物料    │
                    └────────────┬───────────────┘
                                 ▼
                    ┌──────────────────────────────┐
                    │      【练】AI话术训练          │
                    │  基于楼盘自动关联训练题库      │
                    │  AI实时反馈 → 能力雷达图      │
                    │  薄弱点专项训练 → 快速提升     │
                    └────────────┬───────────────┘
                                 ▼
                    ┌──────────────────────────────┐
                    │      【推】新媒体获客          │
                    │  AI生成小红书/公众号文案       │
                    │  内容审核 → 发布管理          │
                    │  推广日历 → 线上引流          │
                    └────────────┬───────────────┘
                                 ▼
                    ┌──────────────────────────────┐
                    │      【约】现场邀约            │
                    │  邀约录音 → AI分析评分        │
                    │  问题识别 → 改进建议          │
                    │  约看率分析 → 持续优化        │
                    └────────────┬───────────────┘
                                 ▼
                    ┌──────────────────────────────┐
                    │      数据沉淀 → 精准营销       │
                    │  客户画像 → 跟进策略优化        │
                    └──────────────────────────────┘
```

### 3.2 各环节核心价值

| 环节 | 核心动作 | 输出成果 | 价值 |
|------|---------|---------|------|
| **探** | 经纪人到楼盘现场 | 真实照片、视频、文字物料 | 标准化探盘流程，确保物料质量统一 |
| **练** | AI话术训练+即时反馈 | 能力雷达图、薄弱点报告 | 快速掌握楼盘卖点，提升经纪人专业度 |
| **推** | AI生成+人工审核发布 | 小红书/公众号高质量内容 | 降低内容生产成本，实现线上获客 |
| **约** | 录音上传+AI分析 | 评分、问题识别、优化建议 | 量化话术能力，提升约看率和成交率 |
